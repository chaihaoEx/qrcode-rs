# qrcode-rs

基于 Rust 开发的二维码生成 Web 应用。

## 功能

- 生成标准 QR Code 二维码
- 提供 Web API 接口
- 高性能、低资源占用

## 快速开始

### 环境要求

- Rust 1.70+
- Cargo

### 构建与运行

```bash
# 构建项目
cargo build --release

# 运行
cargo run
```

## 项目结构

```
qrcode-rs/
├── Cargo.toml       # 项目依赖配置
├── src/
│   └── main.rs      # 程序入口
└── README.md
```
